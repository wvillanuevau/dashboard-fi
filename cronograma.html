Fechas sugeridas calculadas automáticamente a partir de dependencias y duración estimada en días calendario (no días hábiles) desde la fecha de inicio del proyecto. Edítalas libremente. El campo Responsable se llena desde la lista de Involucrados de este mismo proyecto — agrega ahí a las personas antes de asignarlas. El código de acceso es una validación simple del navegador, no reemplaza un control de seguridad real. El respaldo JSON incluye todos los proyectos e involucrados: es la forma de compartir avances entre el equipo en una página estática.
